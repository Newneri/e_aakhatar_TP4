# TP4 C Avancée

Ce projet est réalisé dans le cadre du cours de C avancée à CY Tech. Il s'agit d'un ensemble d'exercices permettant de mettre en pratique les concepts avancés du langage C.

## Structure du projet

Le projet contient plusieurs fichiers sources et un Makefile pour faciliter la compilation et l'exécution des programmes.

## Compilation

Pour compiler le projet, utilisez la commande suivante dans le terminal :

```bash
make 
```

Cette commande va compiler tous les fichiers nécessaires et générer les exécutables.

## Exécution

Après la compilation, vous pouvez exécuter le programme principal avec :

```bash
make run
```

## Nettoyage

Pour supprimer les fichiers générés lors de la compilation, utilisez :

```bash
make clean
```

Cela permet de garder le répertoire propre.

## Documentation

Pour génerer la documentation Doxygene, utilisez la commande suivante dans le terminal :

```bash
make doc
```

Puis pour accéder à la documentation ouvrez le fichier "index.html" qui se trouve dans ./doc/html dans votre navigateur de préference

Pour supprimer la documentation génerée, utilisez la commande suivante dans le terminal :

```bash
make clean-doc
```
